# Codex Skills

Codex Skills maintained by [fushanyx](https://github.com/fushanyx).

## Available skills

### prompt-refiner

Analyzes a frontend project and turns vague development requests into precise,
structured implementation prompts.

Install it by asking Codex:

```text
Install the prompt-refiner skill from:
https://github.com/fushanyx/codex-skills/tree/main/skills/prompt-refiner
```

The skill becomes available on the next turn after installation.

To update it, ask Codex to back up the installed copy and reinstall from the
same URL.
